# Notas Pro 🐾

Una extensión moderna y minimalista para Chromium (Chrome, Edge, Opera) que permite gestionar tus notas de forma eficiente con sincronización en la nube y OCR integrado.

## ✨ Características

- 📝 **Gestión de Notas**: Crea, edita y organiza tus notas con una interfaz premium.
- 🌓 **Temas Dinámicos**: Soporte para modo claro, oscuro y automático.
- ☁️ **Sincronización con Google Drive**: Mantén tus notas seguras y sincronizadas en todos tus dispositivos.
- 🔍 **OCR Selectivo**: Extrae texto de cualquier parte de la pantalla directamente a tus notas.
- 📌 **Panel Lateral**: Usa la extensión como un panel persistente para mayor productividad.

## 🚀 Instalación (Modo Desarrollador)

1. Descarga o clona este repositorio.
2. Abre tu navegador basado en Chromium.
3. Ve a la sección de extensiones (`chrome://extensions`, `edge://extensions` u `opera://extensions`).
4. Activa el **Modo Desarrollador**.
5. Haz clic en **Cargar descomprimida** (Load unpacked).
6. Selecciona la carpeta raíz de este proyecto.

## 🛠️ Tecnologías

- HTML5 / CSS3 (Variables, Animaciones, Flexbox/Grid)
- JavaScript (ES6 Modules, Service Workers, Offscreen Documents)
- Google Drive API v3
- OCR.space API
